# NewVilla

NewVilla is a Real estate Platform (an online platform where people can add and buy real estate properties) using the MERN stack.


## Tech Stack
- React Router DOM
- React Helmet Async
- React Icon
- React-Toastify
- Swiper
- React Rating

## UI Design
- Tailwind CSS
 - Flowbite

## Authentication
- Firebase

## Backend
- Express 
- MongoDB
